# Jenkins CLI

## How to

Clone the project, install python dependencies and run it:

```bash
~/$ git clone git@github.com:alexrochas/jenkins-cli.git
```

```bash
~/jenkins-cli/$ pip3 install requirements.txt
```

```bash
~/jenkins-cli/$ python3 cli.py
```
